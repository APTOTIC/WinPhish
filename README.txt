WinPhish
============
A Simple Windows Form That Writes Typed Password to txt file
In The C Drive under "C:\TmpFls\tmps.txt"

The Installation Path Is "C:\TmpFls"

Requirements
=============
Winx64
winrar
Visual Studio



Instructions
=============
Download Zip File and extract VS Project With Winrar
1) Open Project With Visual Studio
2) Change Username From Aptotic To Vics Username
3) Publish and compress to exe with Winrar
4) Place In Hidden Place On Vics Computer
5) Execute WinPhish.exe
6) Wait for victim to login to Fake Login Page
7) Visit txt File When Victim is Gone
